## Installation

```bash
# install vuepress globally
yarn global add vuepress@next # OR npm install -g vuepress@next

# clone or download repository

# install dependencies
yarn # OR npm install

# start writing
yarn develop

# build
yarn build

# upgrade dependencies
yarn upgrade --latest
```
